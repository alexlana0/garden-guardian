# 🌱 Code Cultivation - Object-Oriented Garden System

Project developed to practice fundamental and advanced **Python** concepts, with a focus on **Object-Oriented Programming (OOP)**.

---

## 📚 About the Project

In this project, we build a "digital garden" management system, where each exercise introduces new features, such as:

- Basic structure of Python programs
- Creating and using classes
- Encapsulation
- Inheritance
- Static and class methods
- Code organization and reuse

---

## 🗂️ Structure

```

.
├── ex0/  # Introduction and basic structure
├── ex1/  # Plant class
├── ex2/  # Growth simulation
├── ex3/  # Plant factory
├── ex4/  # Security and encapsulation
├── ex5/  # Specialized types (inheritance)
├── ex6/  # Analytics and statistics

````

---

## ▶️ How to Run

Make sure you have **Python 3.10+** installed.

To run an exercise:

```bash
python3 ex0/ft_garden_intro.py
````

Or, if using shebang:

```bash
chmod +x ex0/ft_garden_intro.py
./ex0/ft_garden_intro.py
```

## ⚙️ Best Practices

* Use of `if __name__ == "__main__":` to control execution
* Reusable and modular code
* Naming conventions (snake_case and PascalCase)
* Type hints
* Directory-based organization
